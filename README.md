# 🚀 AI Agent for Google Cloud Updates (ADK + MCP)

## 📌 Overview
This project is an AI agent built using Google Agent Development Kit (ADK) and Model Context Protocol (MCP).

The agent connects to BigQuery using MCP Toolbox and fetches real-time Google Cloud release notes, then presents them in a structured format.

---

## 🎯 Problem Statement
Manually tracking Google Cloud updates is time-consuming and inefficient.

This project solves this by building an AI agent that automatically retrieves and summarizes the latest updates.

---

## ⚙️ Tech Stack
- Python
- Google ADK (Agent Development Kit)
- MCP Toolbox
- BigQuery
- Google Cloud Shell

---

## 🔄 System Architecture

User → ADK Agent → MCP Toolbox → BigQuery → Response

---

## ✨ Features
- Fetch latest Google Cloud release notes
- AI-powered responses
- Integration with BigQuery via MCP
- Scalable and modular architecture

---

## 🧠 How It Works
1. User asks for latest updates
2. ADK agent processes the query
3. MCP tool connects to BigQuery
4. SQL query fetches release notes
5. Agent formats and returns response
